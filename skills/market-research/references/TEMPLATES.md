## Pricing Strategy Comparison

```
### Pricing Model Analysis

| Model | Description | When Used | Examples |
|-------|-------------|-----------|----------|
| Freemium | Free basic tier + paid premium | Mass market, viral growth | Slack, Dropbox |
| Subscription | Monthly/annual recurring | SaaS, ongoing value | Netflix, Salesforce |
| Usage-based | Pay per use/unit | Variable consumption | AWS, Twilio |
| Per-seat | Price per user | Team collaboration | Figma, Notion |
| Tiered | Multiple packages | Different segments | HubSpot, Mailchimp |
| One-time | Single purchase | Simple products | Traditional software |
```
